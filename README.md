# Floral GL

Welcome to the **Floral GL** project repository. This collection showcases a series of interactive, WebGL-based visual experiments that render dynamic, blooming floral patterns directly within the browser. The project is designed to explore the intersection of creative coding, mathematics, and performant web graphics.

## Project Structure

This repository is structured as an NPM workspace containing multiple distinct iterations of the core concept. Each iteration explores different visual styles and technical approaches.

- **`iteration-1`**: The inaugural implementation, focusing on interactive bloom animations triggered by user input.
- **`iteration-2`**: A refined approach offering alternative rendering techniques and visual aesthetics.

## Local Development

This repository utilizes npm workspaces to streamline the local development experience. You can install dependencies for all iterations simultaneously from the root directory.

### Prerequisites
- Node.js (v16 or higher recommended)

### Installation
Clone the repository and install the required dependencies:
```bash
npm install
```

### Starting the Development Servers
You can launch the local Vite development server for a specific iteration using the provided npm scripts:

**Run Iteration 1:**
```bash
npm run dev:1
```

**Run Iteration 2:**
```bash
npm run dev:2
```
