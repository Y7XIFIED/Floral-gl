# floral-gl

An interactive WebGL canvas that blooms with flowers when clicked or touched!

## Live Demo
[View on Vercel](https://floral-gl.vercel.app/)

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
